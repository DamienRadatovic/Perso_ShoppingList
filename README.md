- [Ionic]

- [Cordova]

## Utilisation

Pour le web :

```
ionic serve
```

Pour avoir la version mobile :

```
ionic cordova build android
```

OU

Récupérer le fichier apk dans :

platforms/android/app/build/outputs/apk/debug/app-debug.apk
